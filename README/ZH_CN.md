## 什么是 Voyager?
Voyager 是一个全新的基于 Web 的概念操作系统。
## 我该怎么使用它？
#### 1. 克隆源代码
   `git clone https://github.com/tuling-nextstep/voyager/`
#### 2. 重新安装 electron  
   `yarn add electron` 或 `pnpm add electron`  
   我们不推荐您使用 npm ，因为它比 yarn 或 pnpm 更慢/不易用。
#### 3. 运行/调试它
   `yarn start` 或 `pnpm start`
